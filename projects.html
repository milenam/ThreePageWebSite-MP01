Intro to Internet Computing - Static Web Site - Fall 2013

Summary

This assignment required me to make a 3-page client-side web site utilizing HTML5, CSS3, and Bootstrap

Analysis

One of the responsibilities of my job is to maintain a web presence for the store, so this assignment seemed like a breeze. When I started gathering my resources, I found out that Bootstrap reached version 3, which I was very happy about. I was ready to learn new stuff!

Post-mortem

As I worked on the project, I realized how much knowing PHP was a crutch. Not using include tags to simplify navigation, header, and footer sections made it slightly more frustrating coding each page one at a time. I could've included PHP, but I wanted to follow the instructions carefully so I could get an A.
